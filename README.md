# 360 Video Analytics Dashboard

## Clone the Repository

```
git clone https://github.com/GugiGanefianto/360-camera-emotion-analysis.git
```

## Installation

```
cd VR360
pip3 install -r requirements.txt
```

## Run Streamlit

```
cd ./app
streamlit run main.py
```
